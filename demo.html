Eight-Bit.js Generator

A CSS/JQuery plugin to display eight bit sprites in pure CSS.

How to Use

Go to the generator and build your eight-bit sprite. Remember the character name. It is how you will be generating your sprite later. Once your are finished with your grid. Click finished and copy the code. Open you version of eight-bit.js and paste the code under the //Add your eigth bit objects here comment

        //Add your Eight Bit Generator Code Here

Copy this code in to you <head> tag

        <script src="/path_to_file/jquery.eightbit.js" ></script>

In your <body> tag add this code and change out the character name for your and the pixel size for how big you would like your sprite.

        <div class="link1"></div>
        <script>
         $('.link').eightbit({
            pixel :1,
            character : 'your_character_name'  
         });
        </script>
Go to the generator
